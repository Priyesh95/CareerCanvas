# CareerCanvas

Transform your resume into a stunning portfolio website in minutes using AI-powered technology.

## Features

- **AI-Powered Parsing**: Intelligent extraction of work history, skills, and accomplishments from your resume
- **Beautiful Templates**: Three professionally designed templates to choose from
  - **Aurora**: Clean, minimal design with white and light green theme
  - **Nebula**: Professional dark theme with grey and green accents
  - **Spark**: Lively and playful theme with vibrant colors and animations
- **Mobile Responsive**: All templates work flawlessly across devices
- **Interactive Navigation**: Smooth scrolling and active section highlighting
- **100% Free**: No cost, no hidden fees

## Tech Stack

- **React 18**: Modern React with hooks
- **React Router v6**: Client-side routing
- **Vite**: Fast build tool and dev server
- **CSS3**: Custom styling with modern CSS features

## Getting Started

### Prerequisites

- Node.js 16+ and npm

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd CareerCanvas
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Templates

### Aurora
A clean and minimal template with a white and light green color scheme. Perfect for professionals and creatives who prefer a subtle, elegant design.

### Nebula
A professional dark theme with grey and green accents. Makes a strong visual impact and is ideal for tech professionals and developers.

### Spark
A lively and playful template with vibrant colors (yellow, orange, pink, purple) and fun animations. Perfect for creative professionals who want to stand out.

## How It Works

1. **Upload Your Resume**: Drop your resume file or paste the text directly
2. **AI Processing**: Smart algorithms parse and structure your data instantly
3. **Download & Deploy**: Export your portfolio and host it anywhere you like
